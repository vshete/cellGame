README.md
This is a simple game of uncoloring the colored check boxes in a grid before the countdown ends. You can change the grid size (square grid) and the number of cell you will uncolor. You will be given N - 1 seconds to uncover N cells.

Stack: Express - Angualr.js
A simple express server has been created to serve the game. No database has been used.

To start using first install the npm packages using npm install --save.
Then start the server with npm start.
The server can be accessed at localhost:3000
